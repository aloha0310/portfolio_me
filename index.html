Навыки

Что я использую в работе

HTML5
Именно он создает каркас вашего сайта или приложения, а пятая версия позволит мне создавать более SEO-оптимизированную структуру вашего продукта
CSS3
Этот язык стилей позволяет мне создавать абсолютно любой внешний вид Вашего сайта или приложения. Все ограничивается только вашей фантазией!
JS
Этот язык программирования позволяет оживить все что угодно: слайдеры, окна, подсказки, вкладки, получение данных от сервера и многое другое
BOOTSTRAP
Это фреймворк, который используется веб-разработчиками для быстрой вёрстки адаптивных дизайнов сайтов и веб-приложений.
PHP
Это распространённый язык программирования общего назначения. С помощью него пользователь может отправлять Вам письма с сайта.
SASS
Это метаязык на основе CSS, предназначенный для увеличения уровня абстракции CSS-кода и упрощения файлов каскадных таблиц стилей. Именно с помощью него я не заставлю Вас долго ждать свой проект
GIT
Распределённая система управления версиями, для отслеживания и ведения истории изменения файлов, в вашем проекте.
AVOCODE
Это графический редактор, позволяет верстальщику получать код элементов прямо с макета. С помощью него я смогу сделать сайт по шаблону дизайнера с точностью до пикселя.
